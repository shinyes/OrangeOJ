# OrangeOJ v0.5.14

发布时间：2026-05-02

## 本次更新

- 删除题目、删除作业、删除训练和移除空间成员时，确认交互统一改为应用内弹窗
- 删除作业和训练时的“是否同步删除关联题目”二次确认不再使用浏览器默认提示框
- 新增可复用确认弹窗组件与确认 hook，后续危险操作可共用同一套交互
- 应用内确认弹窗支持危险操作按钮颜色、自定义确认文案和取消文案
- 同步更新内置前端构建入口

## 验证

- `frontend`: `npm run build`
