# OrangeOJ v0.5.16

发布时间：2026-05-10

## 本次更新

- 缩小作业提交记录卡片高度，修复提交记录过多时侧边栏不滚动导致题号区域被推出屏幕的问题
- 作业提交后自动清理本地草稿与作答状态，可立即重新做题
- 侧边栏增加最大高度与滚动，确保题号导航在任意记录数量下均可访问

## 验证

- `frontend`: `npm run build`

## 发布说明

本版本发布镜像：

- 主服务镜像：`ghcr.io/shinyes/orangeoj:v0.5.16`
- 判题服务镜像：`ghcr.io/shinyes/orangeoj-judge:v0.5.16`
