# OrangeOJ v0.5.12

发布时间：2026-05-01

## 本次更新

- 修复作业试卷模式中题目按题型分组后仍使用原始作业顺序编号，导致题号跳号的问题
- 试卷模式的题号导航和题面编号现在按最终展示顺序重新连续编号
- 题单模式继续按作业自定义题目顺序展示和编号，不改变已有题单行为
- 同步更新内置前端构建入口，确保后端静态资源加载新版页面

## 验证

- `frontend`: `npm run build`

本版本发布镜像：

- 主服务镜像：`ghcr.io/shinyes/orangeoj:v0.5.12`
- 判题服务镜像：`ghcr.io/shinyes/orangeoj-judge:v0.5.12`
