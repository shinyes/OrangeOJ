# OrangeOJ v0.5.13

发布时间：2026-05-02

## 本次更新

- 题库设置中删除题目时增加确认删除交互，降低误删风险
- 删除作业时可选择是否同步删除关联题目
- 删除训练时可选择是否同步删除关联题目
- 同步删除关联题目时，优先保留仍被其他作业或训练引用的题目
- 提交记录不再阻止同步删题；删除题目时会连带清理提交记录、进度记录和作业提交记录项
- 首页顶栏移除空间选择旁的“管理”快捷按钮
- 登录页不再默认预填 `admin` 用户名
- 同步更新内置前端构建入口

## 验证

- `backend`: `go test ./...`
- `frontend`: `npm run build`

## 发布说明

本版本发布镜像：

- 判题服务镜像：`ghcr.io/shinyes/orangeoj-judge:v0.5.13`
