# OrangeOJ v0.5.1

本次版本聚焦题库、训练、作业和提交记录的完整闭环。

## 主要更新

- 根题库与空间题目支持结构化创建、编辑、删除。
- 训练支持公开/隐藏、自助加入、成员分配、章节化训练页和搜索选题。
- 作业支持试卷模式与题单模式、拖拽排序、独立作业页、独立编程页和提交记录页。
- 作业提交记录支持整卷回看，点击记录可直接回看当次客观题答案与编程题代码。
- 编程题提交记录支持多测试点展示，可查看每个测试点的输入、输出、预期输出和错误信息。

## 验证

- `backend`: `go test ./...`
- `frontend`: `npm run build`
