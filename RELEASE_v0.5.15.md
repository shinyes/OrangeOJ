# OrangeOJ v0.5.15

发布时间：2026-05-07

## 本次更新

- 本地空间选择、题目代码草稿和作业草稿改为按账号隔离保存
- 切换账号后不再读取其他账号遗留的本地草稿或空间选择
- 作业和训练列表增加最大阅读宽度，宽屏下不再横向铺满
- 题库、作业和训练的空状态去掉外框，改为轻量文字提示
- 加宽题库搜索框，完整显示占位文本
- 修复空间下拉框圆角处白色背景溢出
- 同步更新内置前端构建入口

## 验证

- `frontend`: `npm run build`

## 发布说明

本版本发布镜像：

- 主服务镜像：`ghcr.io/shinyes/orangeoj:v0.5.15`
- 判题服务镜像：`ghcr.io/shinyes/orangeoj-judge:v0.5.15`
